# 🚗 Used Car Market Analysis

A comprehensive data analysis project exploring trends, pricing factors, and market patterns in used car listings. This project demonstrates data cleaning, exploratory data analysis (EDA), and visualization using Python.

## 📌 Project Overview

The used car market contains valuable insights into vehicle pricing, fuel preferences, manufacturing trends, and brand popularity. This project analyzes a dataset of used car listings to uncover patterns that influence vehicle prices and consumer preferences.

The analysis includes:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Statistical summaries
* Data visualization

---

## 🎯 Objectives

* Clean and prepare raw vehicle listing data.
* Identify factors affecting used car prices.
* Analyze relationships between vehicle age and price.
* Explore fuel-type distributions.
* Examine brand popularity and market share.
* Create visualizations to communicate findings effectively.


